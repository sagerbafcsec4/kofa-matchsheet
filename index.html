    3対戦リスト（何試合でも）

    #開催日KOホームアウェイ スタジアム②テンプレート
    まとめて貼り付けて追加（1行に1試合）

    「ホーム ［タブ/カンマ］ アウェイ ［タブ/カンマ］ 開催日(任意) ［タブ/カンマ］ KO(任意)」を1行に1試合で。

    4ファイル名の書式

    { } で囲んだトークンがファイル名に入ります（例：{HOME_TEAM}）。一覧は「プレースホルダ」タブ。拡張子.xlsxは自動付与。

    ※ 2試合以上はZIP、1試合はそのままExcel。テンプレートが複数あるときは各行で使うテンプレを選べます（空欄＝先頭）。

    チーム登録

    ホームスタジアムを①②に登録（②は別名や代替名用・任意）。対戦リストで入力したチーム名がここの「チーム名」と一致すると、スタジアム・都市・国が自動で入ります。

    CSVの列順：チーム名, 国, 都市, ホームスタジアム①, ホームスタジアム②（1行目の見出しは任意）。読み込むと既存に追加されます。

    チーム名 国 都市ホームスタジアム①ホームスタジアム②

    使い方

    1) フォーマットベース(.xlsx)のセルに下のプレースホルダを書く → 2) このアプリに読み込む → 3) 共通設定と対戦リストを入力 → 4) まとめてダウンロード。書式・結合セル・複数シートはそのまま保持されます。
